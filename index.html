							HTML里所有的标签一共只分为两种大类：

							非可视化标签：标签不能在网页的内容区域所看见
								如：head、title、meta、script、style等
							可视化标签：标签可以在网页的内容区域所看见
								如：html、body、div、img等等

							对标签：

								简单的说就是成对出现的标签，例如我们之前学到的<title></title>就是对标签
								基本语法：
									<标签名 属性名="属性值" 属性名="属性值">内容>/标签名>

 							

							单标签：

								常用的单标签有：
									<link> 链接引入外部资源标签。
									<img> 向网页中嵌入一幅图像。
									<hr> 标签在 HTML 页面中创建一条水平线。
									建议所有的单标签都书写为<hr />的方式
									理解方式：某些标签是不可能包含文本内容和子标签的，这些标签都是单标签.
									head包含：
										title, link,  meta,  script,  style等常用标签。
							
 							
	
							网页编码：

								网页编码就是对字符串的保存方式
								常见编码：
									ANSI、GBK、GB2312、UTF-8、GB18030和UNICODE
								一般网页编码放在html网页中的<head>和</head>中间。
								<meta?charset= " utf-8" />
								通过改变charset=utf-8中的utf-8就可以改变网页的编码。

 							

							图片标签：

								<img>标签将其他文件夹或服务器的图片显示到网页中
								语法：
								<img src=“图片地址” alt=“自定义替代文本">
								<img>标签只包含属性，并且没有闭合标签

 							

							iframe框架语法

								<iframe src="路径"></iframe>
								"路径"指iframe框架内显示的内容地址
								height,width属性可以规定iframe的高度和宽度(百分比和具体像素都可以)
								<iframe src="路径" width="500px" height="500px">
								</iframe>
								注:再对iframe的高度使用百分比的时候,需要将html,body的高度设置为100%

								frameborder=0			框架边框设置为0即可去除边框
								marginheight=pixels		定义 iframe 的顶部和底部的边距。
								marginwidth=pixels		定义 iframe 的左侧和右侧的边距。
								name="bkName..."		规定 iframe 的名称。
								scrolling=yes/no/auto	规定是否在 iframe 中显示滚动条。
								seamless=seamless		规定 <iframe> 看上去像是包含文档的一部分。
								srcdoc=HTML_code		规定在 <iframe> 中显示的页面的 HTML 内容。
							

							div标签：

								定义文档中的分区或节，有时候用它来进行分层或者布局，因此经常有人说网页布局用Div+CSS，
								其中的Div就是指div标签
								语法结构：
									<div style="color:#00FF00"> 
									<h3>This is a header</h3>
									<p>This is a paragraph.</p>
									</div>

 							

							ul标签：

								定义无序列表
								语法：
									<ul>
										<li>俺是第一个</li>
										<li>俺是第二个</li>
										<li>俺是第三个</li>
									</ul>

 							

							ol标签：

								定义有序列表
								语法：
									<ol>
										<li>one</li>
										<li>two</li>
										<li>three</li>
									</ol>

 							

							dl标签：

								定义列表
								语法：
									<dl>
										<dt>帅哥</dt>
										<dd>蓝桥</dd>
										<dt>美女</dt>
										<dd>班长波</dd>
									</dl>
								<dt>:定义列表中的项目   
								<dd>:描述列表中的项目

 							

							table标签：
								定义HTML表格
								语法：
									<table>
										<tr>
											<th>姓名</th>
											<th>性别</th>
										</tr>
										<tr>
											<td>小明</td>
											<td>男</td>
										</tr>
									</table>

								<tr>定义表格行
								<th>定义表头
								<td>定义表格列
								<table>	定义表格
								<caption>	定义表格标题。
								<th>	定义表格的表头。
								<tr>	定义表格的行。
								<td>	定义表格单元。
								<thead>	定义表格的页眉。
								<tbody>	定义表格的主体.
								border-collapse  collapse合并边框

 							

							frameset标签：

								定义一个框架集
								语法：
									<frameset  cols/rows=“25%,50%,25%>
										<frame src=“url1”/>
										<frame src=“url2”/>
										<frame src=“url3”/>
									</frameset>
								假如一个框架有可见边框，用户可以拖动边框来改变它的大小。
								为了避免这种情况发生，可以在 <frame> 
								标签中加入：noresize="noresize"。
								注意!不可放入body

 							
	
							什么是表单?

								表单是一个包含表单元素的区域,表单元素是允许用户在表单中
								（比如：文本域、下拉列表、单选框、复选框等等）输入信息的元素。
								使用表单标签（<form>）定义。
								<from>
									<p>用户名:<input type="text" name="username" /></p>
									<p>密码:<input type="text" name="password" /></p>
								</from>

								<form action="提交地址" method="get/post">  
									<p>用户名:<input type="text" name="username" /></p>
									<p>密码:<input type="text" name="password" /></p>
									<input type="submit" value="登陆" />
								</form>

								当用户点击按钮提交数据以后,表单内容会被提交到指定文件,该文件对接受的数据进行处理.
								get方式提交
								不安全，所有的参数都会暴露在url里面、url长度有限制
								post提交方式
								安全性高、没有长度限制

 
								input标签：

									<input> 用于搜集用户信息。
									input的作用主要取决里该标签里面的属性,例如我们刚刚讲到的单行文本
									框和密码域的属性分别为type="text"和type="password"
									根据不同的 type 属性值，输入字段拥有很多种形式。
									输入字段可以是文本字段、复选框、掩码后的文本控件、单选按钮、按钮等等。

 
								单行文本框：

									在html页面创建文本域，用户可以在文本域写入文本。
										语法：<input type=“text” name=“自定义名字”>

									密码域，当您在密码域中键入字符时，浏览器将使用项目符号来代替这些字符。
										语法：<input type=“password” name="自定义名字">

								 
								单选按钮：

									当用户点击一个单选按钮时，该按钮会变为选中状态，其他所有按钮都会变为非选中状态。
										语法：
											<input  type=“radio” name=“自定义名字” value=“提交给后台的值”>
										注意：有多个按钮时
										所有按钮的name必须相同。
								 
								复选框：

									可以选择多项
										语法：
											<input type=“checkbox”name=“自定义名字”
										value=“传给后台的值>
										注意：name值必须一样

								 
								文件域：

									文件域表单由一个文本域和一个按钮组成，单击按钮后会激活一个文件选择对话框，
									从本地磁盘中选择一个文件，被选中文件的路径极文件名自动填写在文本域中。
									上传文件时，需要用到文件域。
									语法：
										<input?name="doc"?type="file"/>

								 
								label标签：

									在表单中有很多不同的元素，我们称之为表单域（如姓名、年龄、爱好等）
									label标签用来通过文本的方式告诉用户你当前填写的表单域是什么字段。
										语法：
											<label>姓名：</label>
									由于多选按钮本身比较小，因此出于用户体验的考虑，通常点击选项的文本就应该代表着选择
									对应的checkbox项，可通过label的for属性关联checkbox的id达到此目的。
									例如：
										<input type=“checkbox”name=“interest”id=“interest” value=“传给后台的值”
										checked><label for=“interest”>写程序</label>
									这样，当点击“写程序”这个label的时候，就会触发checkbox的选中
									注意：<label> 标签的 for 属性应当与相关元素的 id 属性相同。

								 
								隐藏域隐藏标签：

									列表表单元素预先把一组可供选择的数据存储在列表中，
									以下拉菜单或列表的形式供用户在其中选择，其中name是选项的提示文字，value才是选项的值。
										语法：
											<input name=”自定义名字” type=”hidden” id=”自定义id” value=”自定义值”/>

								 
								按钮标签：

									定义一个按钮
									语法：
										<input type=“button” value=“自定义显示内容”>（推荐使用）
									或者：
										<button type=“button”>自定义显示内容</button>（不推荐使用）
									注意： form表单元素中的按钮，使用input（type=button)，form之外的按钮可用button，否则会自动提交
									（可以理解为在form中的button默认的type是submit）

								 
								提交：向服务器传送信息

									语法：
										<form>
											<input name="submit" type="submit" value="提交">
										</form>

								 
								重置：

									清除表单中的所有数据
										语法：
										<ltform>
											<input name=“reset” type=“reset” value="重置">
										</form>

								 
								下拉列表：

									语法：
									<select   name=“fruit”>
										<option value=“apple”>苹果</option> 	
										<option value=“pear”>梨子</option>
										<option value=“banana”>香蕉</option<
										<option value=“peach”>桃子</option>
									</select>

									name  规定下拉列表的名称。
									size="" 规定下拉列表中可见选项的数目。
									disabled=disabled  规定禁用该下拉列表。

								 
								文本域：


									<textarea> 定义多行的文本输入控件，文本区中可容纳无限数量的文本，其中的文本的默认字体是等宽字体。通过rows和cols属性设置大小。
									语法：
										<textarea rows="" cols="">
											…
											文本内容
											…
										</textarea>
										


								密码域


									密码域：当您在密码域中键入字符时，浏览器将使用项目符号来代替这些字符。
									用户名:<input type="text" name="username" />
									密码:<input type="password" name="password"/>
									


										在内框架中使用<name>="创建名称"，在外面 使用<a>标签 target="创建名称"可以是页面跳转到指定的网页

										例：
										<img src="" target="">

							规定弹性盒子
								display:flex				给父元素为弹性盒子
								display:box				诶父元素加老的弹性盒子
								display:inline-flex			声明内联弹性盒子 具体什么意思未验证
								display:-webkit-box			兼容问题
						
							规定子元素显示方向:
								flex-direction:row			子元素从左往右显示 也是默认值
								flex-direction:row-reverse		子元素从右往左显示

								flex-direction:column			子元素从上往下显示 
								flex-direction:column-reverse		子元素从下往上显示
							
								flex-wrap:nowrap			让子元素不换行
								flex-wrap:wrap				让子元素换行
								flex-wrap:wrap-reverse			让子元素反转换行

								总结:flex-direction属性,验证后像是数组中的对象,根据自己的需求 改变其排序,排序的是子元素的顺序的位置与定位无关
								注意:而column(上下)和column-reverse(下上)改变的不仅仅是排序更是改变了结构!! 从而影响了justify(X轴)和align(Y轴)位置的互换
								     
								

							复合属性
								flex-flow: flex-direction flex-wrap; 	

							规定子元素以X轴的位置(左,中,右)
								justify-content:flex-start		子元素以X轴左边显示 和float:left类似
								justify-content:center			子元素以X轴中间显示 验证得知 当子元素有固定宽高 会排列在一起在中间显示
								justify-content:flex-end		子元素以X轴右边显示 和float:right类似

								justify-content:space-between		验证得知 当子元素1个的时候是在左边 2个子元素就是左右 3个子元素是左中右显示 相同的margin距离
								justify-content:space-around		验证得知,左右空隙是中间margin的一半 类似于margin控制块级标签与块级标签之间的距离

								总结:justify-content属性,集合了类似于float浮动属性的功能,比float更强大与便捷性
